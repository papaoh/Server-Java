# 서버구축 및 응용
서버구축 및 응용 수업에 만든 게시판 프로그램
- 개발 IDE는 Eclipse를 사용, Tomcat 7.0을 사용하여 서버의 역활을 담당 
- Mysql 라이브러리와 파일업로드 라이브러리를 활용하여 DB까지 연동해 게시판 구축
